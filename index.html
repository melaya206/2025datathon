Data Details

Key columns used in the analysis include:

  • Created Date - When the service request was created
  • Neighborhood - Seattle neighborhood for the request location
  • Council District - City council district for the request
  • ZIP Code - Postal code for the request location
  • Service Request Type - The type of service requested
  • City Department - Department responsible for the request
  • Latitude/Longitude - Geographic coordinates
  • Method Received - How the request was submitted
  • Status - Current status of the request

Refer to the metadata document for full details.
